📌 What is AutoHotkey?

AutoHotkey is a tool for Windows. It lets you create small programs called scripts. These scripts automate repetitive tasks. You can use it to map mouse clicks to keyboard keys. You can also use it to type long phrases with short text triggers. It works on Windows 10 and Windows 11.

🛠️ How to Create Your First Script

Once you install the program, you can create your own automation files. Follow this process to build your first command.
